# ADCH Guidelines Knowledge Pack

This folder contains the ADCH reference material used to build and validate Forgotten Felines policies and compliance evidence.

## What's Here

- `minimum-welfare-operational-standards.md` - ADCH standards baseline
- `adch-standards-principles.md` - principles summary
- `sop1-new-member-applications.md` - application process
- `sop2-existing-member-self-assessment.md` - annual self-assessment cycle
- `sop3-existing-member-external-assessment.md` - external assessment cycle
- `sop4-multi-site-assessment.md` - multi-site assessment rules
- `sop7-non-compliance.md` - non-compliance handling
- `sop8-standards-revision.md` - standards maintenance process
- `adch-critical-standards-quick-reference.md` - practical shortlist of high-impact standards
- `adch-evidence-pack-checklist.md` - what evidence to collect and where it should live
- `archive/` - source PDFs

## How To Use

1. Start with `adch-critical-standards-quick-reference.md`.
2. Use `adch-evidence-pack-checklist.md` to build an assessment-ready evidence set.
3. Cross-check progress in `docs/forgotten-felines/checklists/adch-compliance-checklist.md`.
