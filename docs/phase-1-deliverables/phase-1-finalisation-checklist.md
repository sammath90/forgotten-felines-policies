## Summary: Critical Path

The minimum viable path to trustee approval and publication is:

```
Block A (assign remaining roles)
  → Block B (draft missing policies: Safeguarding, Incident Reporting, Fostering Agreement)
    → Block C (trustee approval session — full animal-care v2.0 suite + registers + Emergency Plan)
      → Block D (publish + distribute)
        → Block E + F (populate systems, deliver training — can run in parallel)
          → Block G (ADCH application)
```

Block E can start in parallel with Block C, since populating registers does not require final trustee sign-off of the underlying policy (only the policy structure).
