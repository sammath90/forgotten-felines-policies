# Veterinary Arrangement Letter — Template

**Forgotten Felines Cat Rescue — Charity No. 1181190**

**Purpose:** a template letter to be sent to each nominated veterinary practice, requesting written confirmation of the services they will provide to Forgotten Felines. The signed returned letter is the evidence required by ADCH Minimum Welfare Operational Standards 13 (formal isolation arrangement) and 51 (nominated vet practice).

**How to use:**

1. Copy the text below into a new document (Google Doc or Word).
2. Replace every `{{placeholder}}` with the real value for the specific practice.
3. Send the letter to the practice with the accompanying cover message.
4. Ask the practice to sign and return it — the practice may print, sign, and scan, or may reply by email confirming every point.
5. File the signed letter in the [Named Veterinary Practices Register](../registers/named-veterinary-practices-register.md) and record the date in the register row for that practice.

The letter must be refreshed whenever the services, primary contact person, or isolation arrangement change — and at least once every 12 months as part of the annual register review.

>
> We would be grateful if you could review the attached letter, confirm the services you can provide to Forgotten Felines, and return a signed copy to us (by scan or return email). It should take a few minutes to read.
>

>

## Filing

Once returned:

1. Save the signed letter (PDF or scan) alongside the Named Veterinary Practices Register.
2. Update the register row for that practice:
   - Set "Arrangement letter on file" to Yes
   - Record the signature date
   - Tick the Services Confirmed boxes that were actually ticked in Section 2
   - If Section 3 confirmed the isolation arrangement, populate the Designated Isolation Practice block
3. Record the exchange in the register's Review Log.
